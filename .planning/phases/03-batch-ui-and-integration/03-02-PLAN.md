**Section 2 — PainelLote.__init__(self, parent):**
- Call `super().__init__(parent, bg=COR_BG)`
- Initialize instance variables: `self._companies = []`, `self._running = False`, `self._elapsed_times = []`, `self._total_companies = 0`, `self._orc = None`, `self._q = None`
- Create StringVars: `self._var_analyst = tk.StringVar()`, `self._var_vigencia = tk.StringVar()`, `self._var_dest = tk.StringVar()`
- Trace the Entry-backed vars (not the Combobox — handle that via event binding):
  `self._var_vigencia.trace_add("write", self._update_start_state)`
  `self._var_dest.trace_add("write", self._update_start_state)`
- Call `self._build_ui()`
- Start poll loop: `self.after(100, self._poll_queue)` — loop runs lifetime of panel, guarded by `self._running` only for reschedule suppression after batch_done (not for stopping permanently)

**Section 4 — Public helper methods (called by tests):**

`_load_analysts_into_combobox(self, names: list[str])`:
- `self._cmb_analyst["values"] = names` (tuple conversion is automatic by ttk)

`_update_start_state(self, *_)`:
- `ok = bool(self._var_analyst.get()) and bool(self._var_vigencia.get().strip()) and bool(self._var_dest.get().strip())`
- `self._btn_start.configure(state="normal" if ok else "disabled")`

`_on_company_start(self, cod: str, i: int, total: int)`:
- `self._total_companies = total`
- `self._pb.configure(maximum=total, value=i + 1)`
- `self._lbl_current.configure(text=f"Processando: {cod}")`

`_on_company_done(self, cod: str, status: str, notes: int, elapsed: float, detail: str)`:
- Append to `self._elapsed_times`
- Compute ETA: `avg = sum(self._elapsed_times) / len(self._elapsed_times)`; `remaining = self._total_companies - len(self._elapsed_times)`; `eta = avg * remaining`
- If remaining > 0: `mins, secs = divmod(int(eta), 60)`; `self._lbl_eta.configure(text=f"ETA: ~{mins}m {secs}s")`
- Else: `self._lbl_eta.configure(text="")`
- Log result: tag based on status; call `self._log(f"[{cod}] {status.upper()} — {detail or f'{notes} nota(s)'}", status if status in ("ok","error","skipped") else "info")`

`_log(self, msg: str, tag: str = "info")`:
- `self._txt_log.configure(state="normal")`
- `self._txt_log.insert(tk.END, msg + "\n", tag)`
- `self._txt_log.see(tk.END)`
- `self._txt_log.configure(state="disabled")`

`_build_summary_text(self, summary: BatchSummary) -> str`:
- Build multi-line string: header line + totals line
- If errors > 0: append "Empresas com erro:" + each CompanyResult where status=="error" as "  • {r.cod} — {r.error_detail}"
- Return string (does NOT call messagebox — keeps it testable)

`_on_batch_done(self, summary: BatchSummary)`:
- `self._running = False`
- Reset buttons: `_btn_start normal`, `_btn_abort disabled`
- `text = self._build_summary_text(summary)`
- Call `messagebox.showwarning(...)` if errors > 0 else `messagebox.showinfo(...)`

**Section 5 — Event handlers:**

`_on_analyst_selected(self, event=None)`:
- `name = self._var_analyst.get()`
- try: `companies = get_companies_for_analyst(name)`; `self._companies = companies`; update `_lbl_count`
- except SpreadsheetError: messagebox.showerror; `self._companies = []`
- call `self._update_start_state()`

`_choose_dest(self)`:
- `path = filedialog.askdirectory(title="Selecionar pasta de destino")`
- if path: `self._var_dest.set(path)`  — StringVar trace fires _update_start_state automatically

`_start_batch(self)`:
- Reset log, reset ETA, reset pb
- `self._q = queue.Queue()`  — fresh queue per batch (pitfall 7)
- `self._orc = BatchOrchestrator(self._q)`
- `self._running = True`
- `self._elapsed_times = []`
- Disable _btn_start, enable _btn_abort
- Read vigencia from _var_vigencia, dest from _var_dest (Path)
- companies = self._companies
- `t = threading.Thread(target=self._orc.run, args=(companies, vigencia, dest), daemon=True)` — daemon=True is Phase 3 responsibility
- `t.start()`

`_abort(self)`:
- if self._orc: `self._orc.abort()`

**Section 6 — Queue poll loop:**

`_poll_queue(self)`:
```python
try:
    while True:
        msg = self._q.get_nowait()
        self._dispatch(msg)
except (queue.Empty, AttributeError):
    pass
self.after(100, self._poll_queue)
```
Note: AttributeError catches the case where `self._q is None` before first batch starts. Loop always reschedules (no stop flag needed — inactive poll is cheap).

`_dispatch(self, msg)`:
```python
kind = msg[0]
if kind == "company_start":
    _, cod, i, total = msg
    self._on_company_start(cod, i, total)
elif kind == "log":
    _, cod, text = msg
    self._log(f"[{cod}] {text}")
elif kind == "counter":
    pass  # optional detail — not surfaced in v1
elif kind == "manual_review":
    _, dados_base, chave_nfse, from_n8n, event, result_holder = msg
    self.after(0, lambda d=dados_base, c=chave_nfse, f=from_n8n, e=event, r=result_holder:
               self._handle_manual_review(d, c, f, e, r))
elif kind == "company_done":
    _, cod, status, notes, elapsed, detail = msg
    self._on_company_done(cod, status, notes, elapsed, detail)
elif kind == "batch_done":
    _, summary = msg
    self._on_batch_done(summary)
```

`_handle_manual_review(self, dados_base, chave_nfse, from_n8n, event, result_holder)`:
```python
try:
    result = abrir_tela_manual_itemlc(
        self.winfo_toplevel(), dados_base, chave_nfse, from_n8n
    )
    result_holder[0] = result
finally:
    event.set()  # ALWAYS set — never leave worker blocked (pitfall 1)
```
